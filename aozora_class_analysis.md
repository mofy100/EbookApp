# 青空文庫オリジナルHTMLのクラス構造分析

代表的な青空文庫のHTMLファイルをスキャンし、頻出するクラスとその役割を整理しました。これらは、パース後のWebビューワーでの表示再現において重要な意味を持ちます。

## 字下げ・配置（ブロック要素）

これらのクラスは、主に `div` タグに付与され、テキストの開始位置や行の長さを制御します。

| クラス名 | 内容・役割 | 処理方針 |
| :--- | :--- | :--- |
| `jisage_X` | X文字分の字下げ（インデント）。左余白を制御。 | `indent-X` またはそのまま保持し、CSSで `margin-left: Xem` を適用。 |
| `jizume_X` | X文字分の字詰め。1行の文字数を制限（Webでは主に幅として機能）。 | 必要に応じて max-width 等にマッピング。 |
| `chitsuki_X` | 地付き（下寄せ、または右寄せ）。X文字分移動。 | `align-end` または末尾寄せスタイルへマッピング。 |

## 見出し・タイトル

作品の構造を表すクラスです。

| クラス名 | 内容・役割 | 処理方針 |
| :--- | :--- | :--- |
| `title` | 作品名（通常 `h1` に付与）。 | `ebook-title-main` へマッピング。 |
| `author` | 著者名（通常 `h2` に付与）。 | `ebook-author-main` へマッピング。 |
| `dai-midashi` | 大見出し。 | 元のレベルを保持しつつ、視覚的にも強調。 |
| `naka-midashi` | 中見出し。 | 同上。 |
| `sho-midashi` | 小見出し。 | 同上。 |
| `midashi_anchor` | 目次用のアンカータグ。 | そのまま保持し、ページ内リンクを維持。 |

## 特殊な表記・装飾

| クラス名 | 内容・役割 | 処理方針 |
| :--- | :--- | :--- |
| `tatechuyoko` | 縦中横（縦書きの中で数字などを横書きにする）。 | CSSの `text-combine-upright` を適用。 |
| `notes` | 注記（返り点や送り仮名、校正注など）。 | 小さめのフォントや点線などで区別。 |
| `gaiji` | 外字画像。標準文字セットにない文字の画像表示。 | 画像パスを解決し、適切なサイズで表示。 |

## その他（削除・無視対象）

| クラス名 | 内容・役割 | 処理方針 |
| :--- | :--- | :--- |
| `main_text` | 本文全体を囲むコンテナ。 | `ebook-body` へ置換。 |
| `bibliographical_information` | 底本情報、入力・校正者の情報（末尾）。 | デフォルトでは非表示、または専用セクションへ。 |
| `notation_notes` | 表記に関する注記（末尾）。 | 同上。 |

---

### まとめ
青空文庫のクラスは `カテゴリ_数値` という命名規則が多く見られます。パース時にはこれらを**そのまま、あるいは規則的に変換して保持**することで、CSSによる一括制御が可能になります。
特に `jisage_` や `chitsuki_` は、縦書きビューワーの再現性において重要です。
